Trabalho realizado com intuito de demonstrar o aprendizado em AWS Step Functions.
O pensamento utilizado foi o de uma loja que armazena todos os dias os fechamentos de vendas diarias, e que no final do dia automaticamente esse arquivo vai para 
uma pasta correta daquele dia e mes em especifico. Utilizei do Lambda para verificação de que o arquivo está realmente na pasta e logo após eu obtive a resposta
correta de que tudo ocorreu como esperado.
